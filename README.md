Пример не коммерческой работы по автоматизации игровых прцессов на основе компьютерного зрения

Описание функционала по папкам:

Fishing_module - Бот автоматизирующий процесс рыбылки, на основе компьютерного зрения (openCV)

Auction_module - Бот автоматизирующий процесс ослеживания и фиксации цен списка интересующих товаров.
Для поиска объектов на экране применяется OpenCV, для распознования цифр - tesseract

hendler - Вспомогательная папка, включающая в себя набор общих функций
